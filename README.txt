Description

Grandpy is a bot that will give you a story for a place you're asking him about. He'll also show you on the map 
where it is, and give you its adress.

How to launch

1. Position yourself in the root folder.
2. Launch virtualenv
3. Launch requirements.txt (pip install -r requirements.txt)
4. Launch app.py (py app.py)
5. Type the adress of the server in your browser

How to test

1. Position yourself in the root folder.
2. Launch pytest
